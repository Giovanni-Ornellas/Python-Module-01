# Python Module 01

# Description

This project consists of the first introduction to Object-Oriented Programming (OOP) in the course. It is composed of a set of seven exercises with progressively increasing difficulty.

The project covers the following topics:

1. Usage of `__name__ == "__main__"`
2. Usage of shebang
3. Classes
4. Getters and Setters
5. `_protected` convention
6. Simple inheritance
7. Static methods
8. Class methods
9. Nested classes

# Instructions

To run the exercises, simply execute the corresponding file.

In addition, all files follow the `flake8` and `mypy` standards, which can be tested using:

> flake8 . && mypy . --strict

ChatGPT was used as an auxiliary research tool whenever certain concepts were not sufficiently clear in the consulted sources or when the available explanations lacked detail.
